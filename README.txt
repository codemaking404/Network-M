NETWORK-M MODULE: QUICK START & SAFETY GUIDE
To use this tool, open a terminal in this folder and run pip install . to load the module and its dependencies (requests, beautifulsoup4). 
Use Download("URL", "flags") to grab files; adding "H" hides the file and "R" runs it immediately—BE EXTREMELY CAREFUL with the "R" flag, as running untrusted code from the web is a major security risk. 
Use TextStrip("URL") to scrape clean, human-readable text from any webpage while filtering out messy HTML code. 
Always ensure Dropbox links end in &raw=1 for direct access, and never use this tool for malicious purposes or on websites that prohibit scraping. Use at your own risk.
